# Scripts Directory

This directory contains project scripts organized as follows:

## Structure

- **boilerplate/** - Template-provided scripts (do not modify)
  - Standard scripts included from the project template
  - Updated when you sync from the template

- **<project-root>/scripts/** - Project-specific scripts (add your custom scripts here)
  - Custom automation scripts for your project
  - Build scripts, deployment helpers, utilities, etc.
  - These scripts are unique to your project

## Usage

- Use boilerplate scripts via Makefile targets: `make init`, `make sync`, etc.
- Add your custom scripts directly in this `scripts/` directory
